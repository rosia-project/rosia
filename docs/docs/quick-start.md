# Quick Start

Let's get started with **Rosia in less than 5 minutes**.

## Installation

```bash
pip install rosia
```

Alternatively, build and install from source:

```bash
git clone https://github.com/rosia-project/rosia.git
cd rosia
pip install -e .
```

Requires `python>=3.9`.
